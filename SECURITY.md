# Security Policy

Report security issues privately to **support@shieldz.cash**, or via the
coordinated-disclosure contact at <https://shieldz.cash/.well-known/security.txt>
(RFC 9116). Please do not open a public issue for security-sensitive reports.

This is the official Rust client SDK for Shieldz. It is non-custodial by
design and never handles private keys or seed phrases — it only sees the public
API key you provide.
